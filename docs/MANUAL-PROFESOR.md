## 🎯 Bienvenido

### ¿Qué es el Sistema Chetango?

El Sistema Chetango es tu herramienta digital para gestionar todo lo relacionado con tus clases de tango. A través de esta plataforma puedes:

✅ **Registrar asistencias** de tus alumnos en cada clase  
✅ **Ver tu calendario** de clases programadas  
✅ **Consultar tus pagos** y estado de liquidaciones mensuales  
✅ **Ver estadísticas** de tus clases y alumnos  
✅ **Actualizar tu perfil** personal  

### ¿Por Qué Usar el Sistema?

- 📱 **Acceso desde cualquier lugar:** Computadora, tablet o celular
- ⏱️ **Ahorra tiempo:** Todo se registra digitalmente, sin papeles
- 💰 **Transparencia:** Ves en tiempo real cuánto has ganado
- 📊 **Mejora tus clases:** Estadísticas de asistencia y progreso

## 🔐 Acceso al Sistema

### Primera Vez: Cómo Ingresar

1. **Obtén tus credenciales:**
   - El administrador te enviará un correo con tu usuario y contraseña
   - Ejemplo: `tu.nombre@corporacionchetango.com`

2. **Ingresa a la plataforma:**
   - Ve a: https://app.corporacionchetango.com
   - Escribe tu correo
   - Escribe tu contraseña
   - Haz clic en **"Iniciar Sesión"**

3. **Primera vez - Cambio de contraseña:**
   - El sistema te pedirá cambiar tu contraseña
   - Elige una contraseña segura que solo tú conozcas
   - Confírmala y guárdala en un lugar seguro

### Acceso Desde el Celular

✅ **Puedes usar el sistema desde tu celular:**
1. Abre el navegador (Chrome, Safari, etc.)
2. Ve a: https://app.corporacionchetango.com
3. Inicia sesión con tus credenciales
4. **TIP:** Guarda la página en tus favoritos para acceso rápido

### ¿Olvidaste tu Contraseña?

1. En la pantalla de inicio de sesión, haz clic en **"Olvidé mi contraseña"**
2. Ingresa tu correo
3. Recibirás un link para crear una nueva contraseña
4. Si no funciona, contacta al administrador

## ✅ Registrar Asistencia de Alumnos

**Esta es tu función principal como profesor.**

### ¿Cuándo Registrar Asistencia?

- Durante la clase o inmediatamente después
- Dentro de las 24 horas (recomendado)
- Antes de que termine el día

### Paso a Paso: Marcar Asistencia

#### 1️⃣ Ve a Registrar Asistencia

```
Menu → Registrar Asistencia
```

#### 2️⃣ Selecciona la Fecha

- Por defecto aparece **HOY**
- Puedes cambiar la fecha usando el calendario
- Solo verás clases que te corresponden

#### 3️⃣ Selecciona tu Clase

El sistema muestra tus clases del día:

```
┌──────────────────────────────────────────┐
│ 🕐 Tango Salón Grupal                    │
│    18:00 - 19:30                         │
│    15 alumnos                            │
│    ✨ [EN CURSO]  ← Si está en horario   │
└──────────────────────────────────────────┘
```

**TIP:** El sistema marca automáticamente la clase que está en curso

#### 4️⃣ Marca Presentes y Ausentes

Verás la lista de alumnos:

```
┌────────────────────────────────────────────────────┐
│  ✅ María García         📦 8 clases disponibles   │
│     Observación: ___________________________       │
│                                                    │
│  ❌ Juan Pérez           📦 3 clases disponibles   │
│     Observación: Llegó tarde                       │
│                                                    │
│  ✅ Ana López            ⚠️  Sin paquete           │
│     Observación: Clase de prueba                   │
└────────────────────────────────────────────────────┘
```

**Leyenda de Estados de Paquete:**
- ✅ `Activo`: Tiene clases disponibles
- ⚠️ `Agotado`: Se le terminaron las clases (avisa al admin)
- 🚫 `Sin Paquete`: No tiene paquete activo (clase de prueba/cortesía)

#### 5️⃣ Agrega Observaciones (Opcional)

Usa el campo "Observación" para notas importantes:
- "Llegó tarde"
- "Excelente progreso"
- "Necesita refuerzo en giros"
- "Primera clase"

#### 6️⃣ Guarda

- Los cambios se guardan **automáticamente** cada vez que marcas presente/ausente
- Verás una notificación verde confirmando el guardado

### Contadores de Asistencia

En la parte superior verás:

```
👥 Presentes: 12    ❌ Ausentes: 3    ⚠️ Alertas: 2
```

**Alertas:** Alumnos con paquete agotado o sin paquete

### ¿Qué Pasa Cuando Marco Presente?

1. **Si el alumno tiene paquete activo:**
   - ✅ Se descuenta 1 clase de su paquete
   - ✅ Se registra la asistencia
   - ✅ Se actualiza el historial del alumno

2. **Si el alumno NO tiene paquete:**
   - ✅ Se registra la asistencia
   - ⚠️ Se marca como "Clase de Prueba" o "Clase de Cortesía"
   - 📢 Se notifica al administrador

### ¿Puedo Editar Después?

✅ **SÍ**, puedes cambiar la asistencia:
1. Vuelve a la misma clase
2. Marca/desmarca presente/ausente
3. El sistema actualiza automáticamente

⚠️ **Importante:** Si cambias de presente a ausente, el sistema **devuelve la clase al paquete** del alumno

### Estados de Pago

- Se generó tu liquidación mensual
- Está listo para pago
- **Acción:** Esperar que te paguen

- Ya recibiste el pago
- Ciclo completado
- **Acción:** Ninguna

### ¿Cómo Funciona el Ciclo de Pago?

```
SEMANA 1-4 DEL MES
├─ Dictas clases normalmente
└─ Registras asistencias

FIN DE MES (Día 28-31)
├─ Admin aprueba todas las clases del mes
├─ Se genera tu Liquidación Mensual
└─ Puedes ver el total a pagar

PRIMERA SEMANA DEL MES SIGUIENTE
├─ Admin realiza el pago (transferencia/efectivo)
├─ Se marca como "Pagado" en el sistema
└─ Ciclo completado ✅
```

### ¿Qué Incluye Mi Pago?

Tu pago mensual se calcula así:

```
TARIFA BASE
    Tarifa por hora × Horas trabajadas × Rol en clase
    
ADICIONALES (si aplican)
    + Bonos por desempeño
    + Horas extras
    + Clases especiales
    
TOTAL A PAGAR
    Base + Adicionales
```

**Ejemplo:**
- Tarifa: $45,000/hora
- Clases grupales (rol principal): 12 clases × 1.5 hrs = 18 horas
- Total: 18 hrs × $45,000 = $810,000

### ¿Dudas sobre tu Pago?

Si ves algo incorrecto:
1. Revisa las clases registradas
2. Verifica las horas trabajadas
3. Contacta al administrador con:
   - Fecha de la clase
   - Observación específica
   - Captura de pantalla (si es posible)

## ❓ Preguntas Frecuentes

### Sobre Asistencias

#### ¿Qué hago si olvidé registrar asistencia?
- Puedes registrarla después seleccionando la fecha pasada
- Recomendado: Hazlo dentro de las 24 horas

#### ¿Puedo cambiar una asistencia ya registrada?
- ✅ SÍ, puedes cambiar de presente a ausente y viceversa
- El sistema ajustará automáticamente los paquetes de los alumnos

#### ¿Qué hago si un alumno no tiene paquete?
- Márcalo presente de todas formas
- El sistema lo registra como "Clase de Prueba"
- El administrador recibirá una alerta
- El alumno deberá comprar un paquete después

#### ¿Puedo registrar asistencia de otra clase que no es mía?
- ❌ NO, solo puedes ver y registrar asistencias de TUS clases

### Sobre Clases

#### ¿Puedo crear o modificar una clase?
- ❌ NO, solo el administrador puede crear/editar clases
- Si necesitas cambios, contacta al administrador

#### ¿Puedo ver clases de otros profesores?
- ❌ NO, solo ves TUS propias clases

#### ¿Qué hago si falta una clase en mi calendario?
- Contacta al administrador inmediatamente
- Puede que no se haya registrado en el sistema

### Sobre Pagos

#### ¿Cuándo me pagan?
- Las liquidaciones se generan a fin de mes
- Los pagos se realizan la primera semana del mes siguiente
- Fechas exactas las define el administrador

#### ¿Puedo ver cuánto llevo ganado en el mes?
- ✅ SÍ, en "Mis Pagos" ves el acumulado del mes actual
- El estado "Pendiente" o "Aprobado" indica en qué fase está

#### ¿Qué hago si mi pago está incorrecto?
1. Verifica las clases registradas en tu historial
2. Compara con tu calendario
3. Contacta al administrador con detalles específicos

#### ¿Por qué algunas clases muestran "Pendiente"?
- Significa que el administrador aún no ha aprobado esa clase
- Es normal durante el mes, se aprueban a fin de mes

### Sobre el Sistema

#### ¿Puedo usar el sistema desde el celular?
- ✅ SÍ, funciona desde cualquier navegador móvil
- Chrome y Safari recomendados

#### ¿El sistema funciona sin internet?
- ❌ NO, necesitas conexión a internet
- Se recomienda WiFi o datos móviles estables

#### ¿Puedo descargar la aplicación?
- No hay app nativa, pero puedes:
  - Guardar el acceso directo en tu celular
  - Usar desde el navegador normalmente
